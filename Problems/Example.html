时间限制：1.00s

空间限制：64.00MB

“C属性大爆发，来吧，竞友们！”

题目描述

游戏《明日方舟》正在进行“促融共竞”活动，在这个活动中，你可以和你的队友一起协作来达成关卡目标。在“堡垒守御”模式的最后，你们需要在 555 分钟内合力击败一个拥有 100w100w100w 血量的敌人。每个人可以携带 101010 名干员，但每个人只能且必须同时部署 666 名干员，每个干员对于该敌人的 DPSDPSDPS （damage per second，每秒伤害）不同。现在由于邪恶兔头力量的影响，敌人血量变成了 101510^{15}1015 ，而限制时间变成了 101010^{10}1010 秒，并且在截止第 ttt 秒内造成的伤害变为了 DPS∗(t+floor(ln(t+DPS))∗105)DPS*(t+floor(ln(t+DPS))*10^5)DPS∗(t+floor(ln(t+DPS))∗105) 而非 DPS∗tDPS*tDPS∗t （为了简化处理，同一个人的干员的总 DPSDPSDPS 合并计算而非对每个干员的 DPSDPSDPS 计算）。现在想询问你最快多少秒内能够将敌人击杀，并计算二人在这段时间内各自造成的伤害（溢出伤害计入，伤害尽可能最大）和各自使用的干员编号。（注： floor()floor()floor() 代表向下取整）

输入格式

第一行输入一个整数 TTT，代表数据组数 ( 1≤T≤1031 \le T \le 10^{3}1≤T≤103 )

对每组数据，第一行输入 101010 个整数 aia_{i}ai​ ( a0a_{0}a0​ ，a1a_{1}a1​ ，...，a9a_{9}a9​)，代表你的 101010 名干员的 DPSDPSDPS，第二行输入 101010 个整数 bib_{i}bi​ ( b0b_{0}b0​ ，b1b_{1}b1​ ，...，b9b_{9}b9​)，代表队友的 101010 名干员的 DPSDPSDPS ( 1≤ai，bi≤1071 \le a_{i} ，b_{i} \le 10^{7}1≤ai​，bi​≤107， iii 即为干员编号)。

输出格式

对每组数据，第一行输出一个整数 ttt，代表最快能在几秒内击杀敌人（若 101010^{10}1010 秒内不能击杀则输出 −1-1−1 ），第二行输出 222 个数，分别代表你造成的总伤害和队友造成的总伤害（若 101010^{10}1010 秒内不能击杀则输出 101010^{10}1010 秒内能造成的伤害），第三行输出 121212 个数，分别代表你使用的 666 个干员编号和队友使用的 666 个干员编号（两组编号各自升序输出）。
